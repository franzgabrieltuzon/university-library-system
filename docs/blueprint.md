# **App Name**: NEU Library Flow

## Core Features:

- Secure Authentication & Role Management: Enable login exclusively with `@neu.edu.ph` Google accounts, implementing distinct 'Regular User' and 'Admin' roles with specific access levels.
- Visitor Check-in System: Provide a user-friendly interface for visitors to record their presence using RFID or their institutional email and select or input their reason for visiting the library.
- Personalized Visitor Welcome: Greet authenticated 'Regular Users' with a customized welcome message, their name, and program upon successful check-in.
- Admin Analytics Dashboard: A secure dashboard for 'Admin' roles displaying comprehensive visitor statistics via informative cards, filterable by date range (daily, weekly, monthly, custom), reason, college, and employee type.
- Visitor Log Management & Export: Allow administrators to search historical visitor logs by name, program, or reason, with the capability to export filtered results as a PDF document. Includes a tool for blocking/unblocking visitor access.
- AI-Powered Visit Reason Insights: An AI tool on the admin dashboard that analyzes and identifies patterns or summarizes common visit reasons from unstructured text inputs, offering enhanced insights into library usage.

## Style Guidelines:

- Primary color: A deep, academic blue (#2752B3) to evoke trust, intelligence, and professionalism, setting a reliable tone for the library system.
- Background color: A very light blue-gray (#F5F7FA) derived from the primary hue, providing a clean, open, and easily readable canvas for all content without visual clutter.
- Accent color: A vibrant, clear cyan (#39C0E3) used for interactive elements and calls to action, ensuring they stand out clearly and add a modern, refreshing touch.
- Headline font: 'PT Sans' (humanist sans-serif) for its modern, yet approachable character, ensuring headings are clear and distinct.
- Body text font: 'Inter' (grotesque-style sans-serif) for its excellent legibility and neutral appearance, perfect for presenting data-rich tables, logs, and detailed information with clarity.
- Use professional, clean-line icons that subtly reflect library themes (e.g., books, reading, computers, charts) for navigation and status indicators, ensuring visual clarity and immediate recognition.
- Employ a modular, card-based layout for dashboards and statistics, promoting content organization and easy scanning of information. Ensure consistent spacing and clear hierarchy.
